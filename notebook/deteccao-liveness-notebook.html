1. Introdução¶

Uma determinada fintech focada em consumidores finais pessoa física constataou um grande número de fraudes em transações bancárias.

O setor de fraudes apontou que existem clientes que se queixaram de não contratar serviços específicos, como o crédito pessoal, e após isso transferir para outras contas desconhecidas.

Após análises pelas equipes de segurança, os protocolos de utilização da senha foram realizados em conformidade, ou seja, cada cliente autenticou com sua própria senha de maneira regular.

Em função disso, o banco precisa arcar com reembolsos e medidas de contenção para evitar processos judiciais, pois os clientes alegam terem sido invadidos por hackers ou algo parecido.

Uma das formas de solucionar ou minimizar este problema é com a utilização de outras formas de autenticação, sobretudo em operações críticas, como a obtenção de crédito pessoal.

Desta forma podemos implementar uma verificação de identidade com prova de vida (liveness), que utilize uma verificação e identificação facial.

Caso o cliente não seja autenticado, ele será atendido por uma esteira dedicada e as evidências da não identificação serão encaminhadas para a área de IA para validação dos parâmetros e limiares para aperfeiçoamento do modelo.

Será necessário construir:

  • Detector de faces
  • Identificação de faces (podendo ser um comparador entre um rosto de documento e outra da prova de vida)
  • Detecção de vivacidade (liveness) para evitar que um fraudador utilize uma foto estática.

Formas alternativas de prover a identificação e prova de vivacidade, além destas que foram solicitadas poderão ser submetidas.

3. Abordagem e organização da solução do problema (2 pontos)¶

Como o grupo pretende deteccar a prova de vivacidade de uma determinada imagem? Quais os passos e os building blocks deste processo?

Resposta:

Para detectar a prova de vivacidade (liveness) de uma determinada imagem, utilizaremos uma abordagem baseada em aprendizado de máquina, especificamente com uma rede neural convolucional (CNN) para classificar as imagens como reais ou fraudulentas. A detecção de vivacidade é essencial para garantir que a imagem apresentada seja de uma pessoa viva e não uma tentativa de fraude utilizando fotos, vídeos ou máscaras.

Passos e Building Blocks do Processo¶

1. Coleta e Preparação dos Dados¶

  • Coleta de Dados: Reunimos um conjunto diversificado de imagens contendo rostos reais (de pessoas vivas) e imagens fraudulentas.
  • Pré-processamento: Normalizamos e padronizamos as imagens, redimensionando para um tamanho uniforme e aplicamos data augmentation (como rotação, espelhamento, e ajustes de brilho) para aumentar a robustez do modelo.
  • Divisão dos Dados: Dividimos o conjunto de dados em conjuntos de treinamento, validação e teste.

2. Detector de Faces¶

  • Implementação: Utilizamos uma biblioteca como OpenCV, dlib ou MTCNN para detectar e extrair regiões de interesse (ROIs) contendo rostos nas imagens.
  • Extração de Faces: Aplicamos o detector de faces para isolar as faces das imagens coletadas, garantindo que apenas as áreas relevantes sejam usadas no treinamento do modelo.

3. Extração de Características Faciais¶

  • Landmarks Faciais: Utilizamos técnicas de landmarks faciais para mapear pontos-chave no rosto, como olhos, nariz e boca.
  • Embeddings Faciais: Geramos embeddings faciais usando redes neurais pré-treinadas como FaceNet ou VGGFace para representar características únicas de cada rosto.

4. Construção do Modelo de Detecção de Vivacidade¶

  • Arquitetura da CNN: Desenvolvemos uma rede neural convolucional (CNN) personalizada ou utilizar uma arquitetura pré-treinada (como ResNet, VGG ou MobileNet) para classificar as imagens.
  • Treinamento do Modelo: Treinamos a CNN com o conjunto de dados preparado, utilizando técnicas de data augmentation para melhorar a generalização do modelo.
  • Classificação de Vivacidade: O modelo deve então classificar cada imagem com uma pontuação de probabilidade indicando a vivacidade (real ou fraudulenta).

5. Avaliação e Ajuste do Modelo¶

  • Métricas de Avaliação: Utilizamos métricas como acurácia, precisão, recall e F1-score para avaliar o desempenho do modelo no conjunto de validação.
  • Ajuste de Hiperparâmetros: Realizamos tuning de hiperparâmetros (como taxa de aprendizado, número de camadas e neurônios) para otimizar o desempenho do modelo.
  • Fine-tuning: Ajustamos a rede neural pré-treinada (se utilizada) para adaptar melhor ao nosso conjunto de dados específico.

# Treinamento, Validação e Teste: Dividir os dados em conjuntos de treinamento (80%), validação (10%) e teste (10%).

import os
import shutil
from sklearn.model_selection import train_test_split

def split_data(input_dir, output_dir, train_ratio=0.8, val_ratio=0.1, test_ratio=0.1):
    # Garantir que os diretórios de saída existam
    for subset in ['train', 'validation', 'test']:
        for class_name in ['liveness', 'fraudulent']:
            os.makedirs(os.path.join(output_dir, subset, class_name), exist_ok=True)
    
    # Listar todas as imagens nas classes
    liveness_images = [os.path.join(input_dir, 'preprocessed_liveness', f) for f in os.listdir(os.path.join(input_dir, 'preprocessed_liveness'))]
    fraudulent_images = [os.path.join(input_dir, 'preprocessed_fraudulent', f) for f in os.listdir(os.path.join(input_dir, 'preprocessed_fraudulent'))]
    
    # Dividir as imagens de 'liveness'
    train_liveness, test_liveness = train_test_split(liveness_images, test_size=(1 - train_ratio))
    val_liveness, test_liveness = train_test_split(test_liveness, test_size=(test_ratio / (val_ratio + test_ratio)))
    
    # Dividir as imagens de 'fraudulent'
    train_fraudulent, test_fraudulent = train_test_split(fraudulent_images, test_size=(1 - train_ratio))
    val_fraudulent, test_fraudulent = train_test_split(test_fraudulent, test_size=(test_ratio / (val_ratio + test_ratio)))
    
    # Função para copiar as imagens para os diretórios de destino
    def copy_images(image_list, dest_dir):
        for image_path in image_list:
            shutil.copy(image_path, dest_dir)
    
    # Copiar as imagens para os diretórios correspondentes
    copy_images(train_liveness, os.path.join(output_dir, 'train', 'liveness'))
    copy_images(val_liveness, os.path.join(output_dir, 'validation', 'liveness'))
    copy_images(test_liveness, os.path.join(output_dir, 'test', 'liveness'))
    
    copy_images(train_fraudulent, os.path.join(output_dir, 'train', 'fraudulent'))
    copy_images(val_fraudulent, os.path.join(output_dir, 'validation', 'fraudulent'))
    copy_images(test_fraudulent, os.path.join(output_dir, 'test', 'fraudulent'))

import dlib

import os
import numpy as np

def detect_faces_and_save(input_dir, output_dir, face_cascade_path='haarcascade_frontalface_default.xml'):
    # Carregar o classificador de cascata para detecção de faces
    face_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + face_cascade_path)

    # Verifica se o diretório de saída existe, se não, cria-o
    if not os.path.exists(output_dir):
        os.makedirs(output_dir)

    for filename in os.listdir(input_dir):
        img_path = os.path.join(input_dir, filename)
        img = cv2.imread(img_path)
        
        if img is not None:
            gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
            faces = face_cascade.detectMultiScale(gray, scaleFactor=1.1, minNeighbors=5, minSize=(30, 30))

            for (x, y, w, h) in faces:
                roi_color = img[y:y+h, x:x+w]
                output_path = os.path.join(output_dir, filename)
                cv2.imwrite(output_path, roi_color)

import dlib
from PIL import Image, ImageDraw

def detect_landmarks(input_dir, output_dir, predictor_path='/Users/bela/computer-vision/shape_predictor_68_face_landmarks.dat'):
    # Carregar o detector de faces e o preditor de landmarks de dlib
    detector = dlib.get_frontal_face_detector()
    predictor = dlib.shape_predictor(predictor_path)

    # Verifica se o diretório de saída existe, se não, cria-o
    if not os.path.exists(output_dir):
        os.makedirs(output_dir)

    for filename in os.listdir(input_dir):
        img_path = os.path.join(input_dir, filename)
        img = Image.open(img_path).convert('RGB')
        img_array = np.array(img)
        
        faces = detector(img_array, 1)
        for i, d in enumerate(faces):
            shape = predictor(img_array, d)
            draw = ImageDraw.Draw(img)
            for j in range(68):
                draw.point((shape.part(j).x, shape.part(j).y), fill=(255, 0, 0))
            output_path = os.path.join(output_dir, f"{os.path.splitext(filename)[0]}_landmarks.jpg")
            img.save(output_path)

import os
import numpy as np
import tensorflow as tf
from tensorflow.keras.preprocessing.image import ImageDataGenerator
from tensorflow.keras.applications import MobileNetV2
from tensorflow.keras.layers import Dense, GlobalAveragePooling2D
from tensorflow.keras.models import Model
from tensorflow.keras.optimizers import Adam

# Diretório de dados
data_dir = "/Users/bela/computer-vision/notebook/imagens/split"


# Carregar e pré-processar os dados
def load_and_preprocess_data(data_dir, subset):
    datagen = ImageDataGenerator(
        rescale=1.0 / 255, shear_range=0.2, zoom_range=0.2, horizontal_flip=True
    )
    generator = datagen.flow_from_directory(
        directory=os.path.join(data_dir, subset),
        target_size=(224, 224),
        batch_size=32,
        class_mode="binary",
    )
    return generator


# Carregar os dados
train_generator = load_and_preprocess_data(data_dir, "train")
validation_generator = load_and_preprocess_data(data_dir, "validation")
test_generator = load_and_preprocess_data(data_dir, "test")

# Construir o modelo
base_model = MobileNetV2(
    weights="imagenet", include_top=False, input_shape=(224, 224, 3)
)
x = base_model.output
x = GlobalAveragePooling2D()(x)
x = Dense(128, activation="relu")(x)
x = Dense(1, activation="sigmoid")(x)  # Saída binária: vivacidade ou fraude

model = Model(inputs=base_model.input, outputs=x)

# Congelar as camadas base do modelo
for layer in base_model.layers:
    layer.trainable = False

# Compilar o modelo
model.compile(
    optimizer=Adam(learning_rate=0.001),
    loss="binary_crossentropy",
    metrics=["accuracy"],
)

# Treinar o modelo
history = model.fit(train_generator, epochs=10, validation_data=validation_generator)

# Descongelar algumas camadas e re-treinar com uma taxa de aprendizado menor
for layer in base_model.layers[-20:]:
    layer.trainable = True

model.compile(
    optimizer=Adam(learning_rate=0.0001),
    loss="binary_crossentropy",
    metrics=["accuracy"],
)

history_fine = model.fit(
    train_generator, epochs=10, validation_data=validation_generator
)

# Avaliar o modelo com o conjunto de teste
loss, accuracy = model.evaluate(test_generator)
print(f"Acurácia no conjunto de teste: {accuracy * 100:.2f}%")

# Salvar o modelo treinado
model.save("liveness_detection_model.h5")

# Carregar o modelo treinado de detecção de vivacidade
liveness_model = load_model('liveness_detection_model.h5')

# Função para detectar e extrair o rosto
def detect_and_extract_face(image_path):
    img = Image.open(image_path).convert('RGB')
    img_array = np.array(img)
    
    faces = detector(img_array, 1)
    for i, d in enumerate(faces):
        x, y, w, h = d.left(), d.top(), d.width(), d.height()
        roi_color = img_array[y:y+h, x:x+w]
        face_img = Image.fromarray(roi_color).resize((224, 224))
        return face_img
    return None

# Função para classificar a vivacidade
def classify_liveness(image_path):
    face_img = detect_and_extract_face(image_path)
    if face_img is None:
        return "Nenhum rosto detectado"
    
    face_array = np.array(face_img) / 255.0
    face_tensor = np.expand_dims(face_array, axis=0)
    
    prediction = liveness_model.predict(face_tensor)
    return prediction[0][0]

# Função para exibir o resultado da classificação
def display_result(image_path):
    prediction = classify_liveness(image_path)
    if prediction == "Nenhum rosto detectado":
        print(prediction)
    else:
        print(f"Pontuação de vivacidade: {prediction}")
        if prediction >= 0.5:
            print("Classificação: Rosto Real")
        else:
            print("Classificação: Rosto Forjado")
    
    img = Image.open(image_path)
    plt.imshow(img)
    plt.axis('off')
    plt.show()
